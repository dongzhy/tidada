# TiDaDa AI 答题测评系统
## 项目简介
TiDaDa 是一套前后端分离全栈 AI 答题测评系统，后端基于 SpringBoot 开发，Web 管理后台采用 Vue3 + TypeScript，微信小程序使用 Taro3 跨端实现。
平台同时支持自定义计分答题、AI 智能性格测评两种业务模式，搭建普通用户 / 管理员两级权限体系，覆盖应用创建、题目管理、AI 批量出题、评分配置、用户答题、应用审核、数据统计完整业务链路。
项目地址：https://github.com/donghaiyong
## 一、技术栈
### 后端技术栈
核心框架：SpringBoot 2.7、SpringMVC、MyBatis-Plus
数据库：MySQL（6 张核心业务数据表）
缓存 & 并发：Caffeine 本地缓存、Redis、Redisson 分布式锁
AI 能力：第三方大模型 SDK接入智谱api、RxJava 异步流处理
权限体系：自定义 @AuthCheck 注解实现接口级权限拦截
工具类：Hutool、EasyExcel（待实现 Excel 批量导入题库）
设计模式：策略模式（双评分策略解耦）
### Web 管理后台
基础框架：Vue3 + TypeScript + Vue-CLI
状态管理：Pinia
UI 组件：Arco Design Vue
工具库：dayjs、ECharts、qrcode
工程规范：ESLint + Prettier、OpenAPI 自动生成接口请求代码

## 二、数据库设计（6 张核心业务表）
- user 用户表：账号、密码、角色、微信 openid、昵称头像、个人简介
- app 应用表：应用名称、描述、图标、应用类型、评分策略、审核四件套（审核状态 / 审核人 / 审核时间 / 驳回理由）
- question 题目表：JSON 存储题目与选项，区分「得分类（带分值）」「测评类（带属性标识）」两套结构
- scoring_result 评分结果表：测评名称、描述、图片、属性数组、得分匹配区间
- user_answer 用户答题记录表：用户所选答案、最终得分 / AI 测评结果、关联应用与用户
- 统计聚合：Mybatis 自定义聚合 SQL，统计答题量、测评结果分布
## 三、后端模块 & 接口说明
1. AppController 应用模块
   - 新增应用：普通用户创建应用，默认置为待审核状态
   - 编辑应用：用户修改自有应用，修改后重置审核状态为待审核
   - 管理员更新：全量编辑应用所有信息
   - 审核接口：管理员审核，支持通过 / 驳回，可填写驳回理由
   分页查询三类接口：
   - 管理员：查询全量应用（含待审核、驳回）
   - 个人：仅查询当前登录用户创建的应用
   - 首页：仅展示审核通过的公开应用
   - 权限控制：普通用户仅操作自身应用，管理员拥有全部审核、管理权限
2. QuestionController 题目模块
   - 基础 CRUD 能力 + AI 生成核心能力
   - 同步 AI 出题：根据应用信息批量生成题目 JSON 数组
   - SSE 流式 AI 出题：长文本逐字符实时推送，消除页面长时间等待
   - 自动适配两类应用：得分类选项携带分值、测评类携带 MBTI 属性标识
3. ScoringResultController 评分结果模块
   - 仅自定义得分类应用需要配置，AI 测评应用无需手动维护评分结果
   - 提供新增、删除、管理员更新、用户编辑、分页列表、详情 VO 接口；自动完成 List 数组与 JSON 字符串序列化转换。
4. UserAnswerController 答题核心模块
   - 基于策略模式实现两套独立打分逻辑
   - 自定义计分策略：累加每题选项分数，匹配最低分数区间返回对应测评结果
   - AI 测评策略：本地缓存 + 分布式锁拦截重复 AI 调用，降低大模型调用成本
   - 用户提交答题后自动执行打分逻辑，回填结果存入答题记录表。
5. AppStatisticController 数据统计模块
   - 自定义聚合 SQL，为前端图表提供数据源：
   - 热门应用答题数量排行
   - 单个应用内各测评结果分布统计
6. UserController 用户模块
   支持账号注册、密码登录、退出登录、获取当前登录用户；
   管理员可新增、分页管理全量用户，普通用户仅能修改个人信息。
### Web 后台页面功能
   首页：展示全部已过审应用，支持搜索、生成分享二维码
   应用管理（管理员）：全应用列表、多条件筛选、批量审核
   我的应用：个人创建应用，快速跳转题目 / 评分配置页面
   题目管理：手动新增、AI 一键批量生成题目
   评分结果管理：得分类配置评分区间（已修复修改按钮点击无响应线上 Bug）
   答题记录：查看所有用户答题完整记录
   数据统计：答题量、测评分布可视化图表
   用户管理（管理员）：平台所有用户增删改查
