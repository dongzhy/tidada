# MBTI 性格测评小程序
基于 Taro3 + React + TypeScript + Sass 开发的轻量化 MBTI 性格测试微信小程序，支持快速答题、自动计算性格类型、展示测评结果，界面简洁、体验流畅。
## 项目简介
mbti-mini-test 是一款轻量型微信小程序，内置完整 MBTI 测试题库，用户完成答题后自动计算性格维度，展示 16 型人格结果与详细描述。
纯前端实现，无需后端接口
完整答题 + 结果计算 + 结果展示闭环
支持答案本地缓存、异常容错处理
基于 Taro UI 组件库快速构建界面
### 技术栈
- 框架：Taro 3.6.28
- 开发语言：React 18 + TypeScript
- 样式：Sass
- UI 组件：Taro UI
- 构建工具：Webpack5
- 运行平台：微信小程序（支持编译多端）


## 核心功能
- 内置 10 道标准性格测试题
- 涵盖 E/I、S/N、T/F、J/P 四大维度
- 自动计算结果
- 统计用户答案维度得分
- 匹配最优 16 型人格结果
- 结果展示页
- 展示人格名称、详细描述
- 异常捕获、空值容错
- 基础交互
- 答案本地存储
- 结果页返回首页
- 错误提示与日志输出

#### 多端编译支持

#### 结果页逻辑
读取本地缓存答案列表
校验答案与 JSON 数据格式
调用 getBestQuestionResult 计算人格结果
异常捕获：答案为空、数据格式错误、计算失败
提供返回主页按钮，支持页面重定向
#### 运行效果
进入小程序 → 开始答题
提交答案 → 自动计算 MBTI 类型
展示结果：人格名称 + 详细描述
